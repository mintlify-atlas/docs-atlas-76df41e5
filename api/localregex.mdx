title: LocalRegex class
description: Core class containing all validation methods for Zimbabwean mobile numbers, VoIP providers, identity documents, and utilities
---

The `LocalRegex` class provides static methods for validating various types of data, with a focus on Zimbabwean telecommunications and identity documents.

## Mobile Network Validation

Methods for validating mobile phone numbers from major Zimbabwean network providers.

### isEconet()

Checks if the given number is a valid Econet mobile number.

<ParamField path="value" type="String" required>
  The phone number to validate (supports formats: 0771234567, +263771234567, 263771234567)
</ParamField>

<ResponseField name="returns" type="bool">
  Returns `true` if the number is a valid Econet number, `false` otherwise
</ResponseField>

```dart
if (LocalRegex.isEconet('0771234567')) {
  print('Valid Econet number');
}

if (LocalRegex.isEconet('+263771234567')) {
  print('Valid Econet number with country code');
}
```

### isPassword()

Checks if the given string is a valid password (requires: uppercase, lowercase, digit, special character, minimum 8 characters).

<ParamField path="password" type="String" required>
  The password to validate
</ParamField>

<ResponseField name="returns" type="bool">
  Returns `true` if the password meets requirements, `false` otherwise
</ResponseField>

```dart
if (LocalRegex.isPassword('MyPass123!')) {
  print('Strong password');
}
```

## Utility Methods

### generatePassword()

Generates a cryptographically secure random password.

<ParamField path="length" type="int" default="12">
  The length of the password to generate
</ParamField>

<ResponseField name="returns" type="String">
  A randomly generated password containing uppercase, lowercase, numbers, and special characters
</ResponseField>

```dart
final password = LocalRegex.generatePassword();
print(password); // e.g., "aB3$xY9!mK2@"

final longPassword = LocalRegex.generatePassword(length: 20);
print(longPassword); // 20-character password
```

<Note>
  The generated password always contains at least one character from each category: lowercase, uppercase, digit, and special character.
</Note>
