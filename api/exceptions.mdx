The `LocalRegexException` class represents exceptions that occur during validation and formatting operations in the LocalRegex library.

## Class Definition

```dart
class LocalRegexException implements Exception {
  LocalRegexException(this.message);
  
  final String message;
  
  @override
  String toString() {
    return 'LocalRegexException: $message';
  }
}
```

## Constructor

### LocalRegexException()

Creates a new instance of `LocalRegexException` with the specified error message.

<ParamField path="message" type="String" required>
  A descriptive error message explaining what went wrong
</ParamField>

```dart
throw LocalRegexException('Phone number is not valid!');

throw LocalRegexException('National ID is not valid');

throw LocalRegexException('Phone number length cannot be less than 9');
```

## Properties

### message

The error message describing the exception.

<ResponseField name="type" type="String">
  A human-readable description of the error
</ResponseField>

```dart
try {
  final formatted = 'invalid'.formatNumber(
    formatType: FormatType.regular,
  );
} on LocalRegexException catch (e) {
  print(e.message); // Phone number is not valid!
}
```

## Common Exception Messages

The library throws `LocalRegexException` with these common messages:

| Message | Cause | Thrown By |
|---------|-------|----------|
| "Phone number is not valid!" | Invalid phone number format | `formatNumber()` |
| "Phone number length cannot be less than 9" | Phone number too short | Internal formatting functions |
| "National ID is not valid" | Invalid national ID format | `formatID()` |

### Form Validation

```dart
class PhoneNumberValidator {
  String? validate(String? value) {
    if (value == null || value.isEmpty) {
      return 'Phone number is required';
    }
    
    try {
      value.formatNumber(formatType: FormatType.regular);
      return null; // Valid
    } on LocalRegexException catch (e) {
      return e.message; // Return error message
    }
  }
}

// In Flutter
```

### ID Formatting with Error Recovery

```dart
String formatIdSafely(String rawId) {
  try {
    return rawId.formatID(formatType: IdFormatType.proper)!;
  } on LocalRegexException catch (e) {
    print('ID formatting error: ${e.message}');
    return rawId; // Return original on error
  }
```

### Batch Processing with Error Logging

```dart
class PhoneNumberProcessor {
  final List<String> errors = [];
  
  List<String> formatAll(List<String> phoneNumbers) {
    final formatted = <String>[];
    
    for (final phone in phoneNumbers) {
      try {
        final result = phone.formatNumber(
          formatType: FormatType.countryCode,
        );
        if (result != null) {
          formatted.add(result);
        }
      } on LocalRegexException catch (e) {
        errors.add('$phone: ${e.message}');
      }
    }
    
    return formatted;
  }
}

// Usage
final processor = PhoneNumberProcessor();
final numbers = ['0771234567', 'invalid', '0711234567', '1234'];
final results = processor.formatAll(numbers);

print('Formatted: ${results.length} numbers');
print('Errors: ${processor.errors.length}');
for (final error in processor.errors) {
  print('  - $error');
}
```

```dart
  final String? value;
```

### Custom Exception Handling

```dart
class FormattingException implements Exception {
  final String field;
  final String message;
  
  FormattingException(this.field, this.message);
  
  @override
  String toString() => 'Error in $field: $message';
}

String formatPhoneOrThrow(String phone, String fieldName) {
  try {
    final result = phone.formatNumber(
      formatType: FormatType.countryCode,
    );
    if (result == null) {
      throw FormattingException(fieldName, 'Formatting returned null');
    }
    return result;
  } on LocalRegexException catch (e) {
    throw FormattingException(fieldName, e.message);
  }
}

// Usage
try {
  final phone = formatPhoneOrThrow(userInput, 'contactNumber');
  await saveToDatabase(phone);
} on FormattingException catch (e) {
  print(e); // Error in contactNumber: Phone number is not valid!
}
```

### Logging and Monitoring

```dart
class PhoneFormatter {
  final Logger logger;
  
  PhoneFormatter(this.logger);
  
  String? format(String phone, FormatType formatType) {
    try {
      final result = phone.formatNumber(formatType: formatType);
      logger.info('Successfully formatted: $phone');
      return result;
    } on LocalRegexException catch (e) {
      logger.error(
        'Failed to format phone number',
        error: e,
        stackTrace: StackTrace.current,
        data: {'input': phone, 'message': e.message},
      );
      return null;
    }
  }
}
```

## Best Practices

### Always Handle Exceptions

```dart
// ❌ BAD: Unhandled exception can crash the app
final formatted = userInput.formatNumber(
  formatType: FormatType.regular,
);

// ✅ GOOD: Proper error handling
try {
  final formatted = userInput.formatNumber(
    formatType: FormatType.regular,
  );
  processNumber(formatted);
} on LocalRegexException catch (e) {
  showUserFriendlyError(e.message);
}
```

### Validate Before Formatting

```dart
// ✅ BEST: Validate first, then format
if (LocalRegex.isZimMobile(userInput)) {
  final formatted = userInput.formatNumber(
    formatType: FormatType.countryCode,
  );
  // No exception will be thrown here
} else {
  showError('Please enter a valid Zimbabwean mobile number');
}
```

### Provide User-Friendly Messages

```dart
String getUserFriendlyMessage(LocalRegexException e) {
  if (e.message.contains('Phone number is not valid')) {
    return 'Please enter a valid Zimbabwean phone number (e.g., 0771234567)';
  } else if (e.message.contains('National ID is not valid')) {
    return 'Please enter a valid national ID (e.g., 63-123456 A 53)';
  } else if (e.message.contains('length cannot be less than 9')) {
    return 'Phone number is too short. Please enter a complete number.';
  }
  return 'Invalid input: ${e.message}';
}
```
